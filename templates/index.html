Features


Live Face Recognition

This Web-Application does real-time face recognition by analyzing the user's face and then matching it with the database of registered users. If the user image matches, Attendance is marked for that user. It also provides the option to register new users.

Fake Attendance Detection

First, the admin provides the location of the place where work/class is going on. When the user enters their account, their device location is recorded and compared with the location point they provide, and this ensures that the user is present at the place of work .

No External Person Or Device Needed

As every user have an account, they have to reach the desired location, log in to their account and mark their attendance. This way, there is no need to stand in long queues neither they need any external machine nor wait for the person to come and mark their attendance.
