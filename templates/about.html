Attendance Tracker

It is a web app that does real-time face recognition, detects false attendance by accessing the user's location, and does not require a machine or human to mark the attendance. Working with it is very much straightforward. Users need to create an account and log in and keep their attendance. It has two options, one for the admin and the user. The admin specifies the location where the work/class is going on, and this web app uses the user location to ensure that the user is present at that place
